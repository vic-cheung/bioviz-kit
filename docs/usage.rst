Usage
=====

Quickstart
----------

Install the package (see README for alternatives):

.. code-block:: bash

   pip install bioviz-kit

Minimal example
--------------

.. code-block:: python

   from bioviz import lineplot
   # fig = lineplot.generate_styled_lineplot(df, config)

Examples
--------

See the `examples/` directory for runnable `.py` scripts and notebooks.

Documentation notes
-------------------

- Each example should include: purpose, required inputs, minimal commands, and expected output.
- Prefer paired `.py` + `.ipynb` via `jupytext` for reproducibility and narrative.
